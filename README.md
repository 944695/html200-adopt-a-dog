# Adopt a Dog (HTML200)

Adopt a Dog is a multi-page HTML website for a dog adoption organization. The project focuses on semantic HTML structure, clear content hierarchy, and basic responsive setup using the viewport meta tag.

## Site Pages

- Home: [index.html](index.html)
- Dogs listing: [dogs.html](dogs.html)
- Blog: [blog.html](blog.html)
- Adoption profile/form: [checkout.html](checkout.html)

## Project Goals

- Build clean, semantic page structure using HTML5 elements.
- Keep navigation consistent across all pages.
- Present adoption content in clear, scannable sections.
- Prepare the site for responsive enhancement in later assignments.

## Highlights

- Semantic elements used throughout (`header`, `nav`, `main`, `section`, `article`, `footer`).
- Dog cards and blog posts organized for readability.
- Adoption form includes labels, grouped sections, and required fields.
- Alt text provided for all content images.
